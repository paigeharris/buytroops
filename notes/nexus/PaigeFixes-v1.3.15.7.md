# PaigeFixes v1.3.15.7

## Synopsis
PaigeFixes is a standalone native-fixes helper mod for Bannerlord and War Sails edge cases. This version protects the civilian-equipment path so childbirth or civilian-outfit generation does not explode when a valid civilian set is missing, and it warns players before using the specific Native/War Sails lift-blockade action that is known to crash in the independent-ruler, no-ships, port-siege scenario.

## Nexus File Description
Adds a civilian-equipment fallback for childbirth NPC crashes and warns on the Native/War Sails lift-blockade action that can crash for independent rulers with no ships.
