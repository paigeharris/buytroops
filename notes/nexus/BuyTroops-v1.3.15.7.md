# BuyTroops v1.3.15.7

## Synopsis
BuyTroops adds retinue-buying menus to towns, castles, and War Sails ports so you can hire themed troop packs directly from campaign menus. This version keeps the gameplay flow the same while reducing noisy debug popups and preserving only the player-facing purchase feedback. It also keeps the existing safety guards that pause or shut down menu actions if Bannerlord is in a crash-prone siege or combat state.

## Nexus File Description
Adds town, castle, and port troop-buying menus, with extra safety guards to avoid siege/combat crash states while keeping purchase feedback visible.
